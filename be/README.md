# Crypto Mock Trading - Backend

Spring Boot application for cryptocurrency mock trading platform with AI coaching.

## Tech Stack
- Java 17
- Spring Boot 3.5.6
- PostgreSQL 15
- Redis 7
- Binance WebSocket API

## Quick Start

```bash
# Start dependencies
docker-compose up -d

# Run application
./mvnw spring-boot:run
```

## Configuration

Database and API settings in `application.properties`.

Environment variables can be set in `.env` file (see `.env.example`).
